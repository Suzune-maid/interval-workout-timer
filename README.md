# 乾式高潮導向訓練儀表板

依據 `dry_orgasm_training_plan.md` 整理成的 6 週專門訓練網站。

## 目前功能

- 以 **2026-04-27** 作為第一週第一天
- 每天依日期自動切換到對應訓練內容
- 顯示今日課表、重點提醒與安全邊界
- 內建今日流程計時器
- 顯示完整 6 週日程表
- 使用 Node 內建 test runner 驗證核心邏輯

## 本機預覽

```bash
python3 -m http.server 8124
```

打開 <http://127.0.0.1:8124/>。

## 測試

```bash
npm test
```

## 部署

此專案部署在 GitHub Pages，方便直接用手機或桌機開啟。
